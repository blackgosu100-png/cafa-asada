네이버 리뷰 복사 버튼 설치

직원은 코드를 볼 필요 없이, 스마트스토어 리뷰 화면에서 즐겨찾기 버튼만 누르면 됩니다.

처음 한 번만 설치

  1. Chrome에서 즐겨찾기 바가 안 보이면 Ctrl + Shift + B를 누릅니다.
  2. 아래 초록 버튼을 마우스로 끌어서 즐겨찾기 바에 놓습니다.
  3. 이후 네이버 스마트스토어 리뷰 화면에서 그 즐겨찾기 버튼을 누르면 됩니다.
리뷰 복사 버튼을 클릭하는 게 아니라, 즐겨찾기 바로 드래그해서 넣는 방식입니다.

매번 사용하는 방법

  1. 스마트스토어 상품 페이지에서 리뷰 창을 엽니다.
  2. 필요한 만큼 아래로 스크롤해서 리뷰를 화면에 로딩합니다.
  3. 즐겨찾기 바의 리뷰 복사를 누릅니다.
  4. 리뷰 내용이 자동으로 클립보드에 복사되고, CSV 파일도 같이 내려받습니다.
  5. 복사된 내용을 게시글 생성기에 붙여넣어 제목과 본문 초안을 만듭니다.
공개로 보이는 리뷰만 복사하는 용도입니다. 고객 아이디, 개인정보, 비공개 정보는 게시글에 넣지 말고, 실제 후기를 특정 고객 발언처럼 꾸미지 않는 방식으로 쓰는 게 좋습니다.
